Standoff Generator

Fully client-side Ender 3 v3 SE/KE standoff generator, in the browser. Powered by OpenSCAD.

Input the corner values of your levelling data into the grid on the right and click Generate!



Front

What is this for?

Some users of Creality Ender 3 v3 SE and Ender 3 v3 KE FDM 3D printers encounter issues with bed levelling with some of the corners being off. This tool helps you correct for that.

How do I print these?

The standoffs should be printed with the smallest layer height your printer can manage with the installed nozzle and in a filament that can withstand the bed temperatures, like ASA.
